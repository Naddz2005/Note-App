# Giới thiệu website/hệ thống

Note App là ứng dụng quản lý ghi chú được phát triển bằng Flutter kết hợp với Firebase Realtime Database và Firebase Authentication.

Hệ thống cho phép người dùng:

- Đăng ký tài khoản
- Đăng nhập hệ thống
- Tạo ghi chú
- Chỉnh sửa ghi chú
- Xóa ghi chú
- Tìm kiếm ghi chú
- Hiển thị ghi chú dạng Grid View và List View
- Quản lý thông tin cá nhân
- Chuyển đổi Dark Mode

Dữ liệu được lưu trữ theo thời gian thực bằng Firebase Realtime Database.

# Hướng dẫn cài đặt

## Bước 1: Cài đặt Flutter

Tải Flutter SDK tại:

https://docs.flutter.dev/install

Kiểm tra Flutter: flutter doctor

## Bước 2: Cài đặt Android Studio

Tải Android Studio tại:

https://developer.android.com/studio

Cài đặt plugin Flutter

## Bước 3: Clone source code: git clone https://github.com/Naddz2005/Note-App

## Bước 4: Cài đặt dependencies

Mở terminal trong project và chạy: flutter pub get

# Hướng dẫn chạy project
Chạy trên điện thoại thật
Bật Developer Options
Bật USB Debugging
Kết nối điện thoại với máy tính
Chạy project: flutter run
